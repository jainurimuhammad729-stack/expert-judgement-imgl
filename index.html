📌 Petunjuk Expert Judgement

Tata Cara Penilaian

  • Baca setiap item instrumen dengan seksama
  • Berikan skor 1–4 pada setiap aspek penilaian
  • Isi kolom saran perbaikan bila diperlukan
  • Tentukan keputusan akhir (DR/R/T) untuk setiap item
  • Isi kesimpulan dan rekomendasi di akhir validasi
  • Klik "Simpan & Ekspor" untuk menyimpan hasil

Skala Penilaian

4Sangat Relevan / Sangat Sesuai / Sangat Jelas
3Relevan / Sesuai / Jelas
2Kurang Relevan / Kurang Sesuai / Kurang Jelas
1Tidak Relevan / Tidak Sesuai / Tidak Jelas

Keputusan Item

  • DR – Dapat Digunakan tanpa Revisi
  • R – Dapat Digunakan dengan Revisi
  • T – Tidak Dapat Digunakan

Target Kualitas (CVI/CVR)

  • CVI ≥ 0,80 → instrumen valid
  • Aiken's V ≥ 0,75 → item diterima
  • Aiken's V 0,60–0,74 → revisi item
  • Aiken's V < 0,60 → item ditolak
  • CVR menggunakan formula Lawshe (1975)

Alur Validasi

  • Isi identitas validator di form login
  • Validasi konstruk instrumen SLEI
  • Validasi item per dimensi SLEI (40 item)
  • Validasi kisi-kisi & item SRAI (50 soal)
  • Validasi instrumen Wawancara
  • Validasi instrumen FGD
  • Isi kesimpulan & ekspor PDF
